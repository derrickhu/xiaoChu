《五行通天塔》完整游戏策划文档（含数值+全系统）

版本：最终可开发版 | 类型：修仙转珠爬塔Roguelike | 无局外养成
一、游戏核心定位

1. 玩法：类智龙迷城五行转珠战斗 + 无限爬塔 + 局内随机成长

2. 核心循环：开局→随机地图事件→转珠战斗/奇遇/商店→替换宠物/武器→挑战更高层

3. 规则：无局外属性、无付费、无等级，死亡即重开，仅记录最高层数

4. 美术风格：国风修仙、简洁Q版/国风写实均可
二、核心战斗系统（转珠规则）

1. 棋盘设定

• 尺寸：6×6 = 36颗珠子

• 珠子类型：金、木、水、火、土、心（回血）

• 操作：手指拖动珠子八方向换位，无限连消，限时8秒

• 松手 = 回合结束 = 怪物攻击1次

2. 伤害与消除规则

• 3消：对应属性宠物正常攻击

• 4消：伤害×1.5

• 5消及以上：全体攻击 + 眩晕怪物1回合

• 无对应属性宠物 → 消除无效，不计算Combo

• Combo加成：每1段Combo → 所有伤害**+25%**

• 心珠：消除即回血，与属性无关

3. 血量与生存

• 仅主角1条血条，宠物不承受伤害

• 主角血量归0 → 爬塔结束

• 回血来源：心珠、技能、武器、奇遇、休息

4. 宠物主动技能触发

• 每完成1次转珠回合 → 所有宠物技能CD-1

• CD=0 → 头像亮起，转珠前点击释放

• 释放后CD重置，重新冷却
三、主角设定

1. 初始血量：100点

2. 上场宠物：固定5只（金木水火土属性）

3. 装备武器：仅1把，提供全队被动/光环效果

4. 无任何局外成长，每局完全重置
四、宠物系统（5属性×20只=100只，含技能+CD+基础攻击）

统一规则

• 宠物仅在消除对应属性珠时攻击

• 爬塔胜利可替换/强化宠物
