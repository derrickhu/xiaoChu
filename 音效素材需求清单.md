# 灵宠消消塔 - 音效素材需求清单

## 整体风格定位

**游戏类型**：修仙主题 × 三消玩法 × 可爱画风  
**音效总基调**：国风 + 可爱 + 灵动  
**参考乐器音色**：古筝拨弦、竹笛、玉石碰撞、风铃、木鱼、小鼓、铃铛  
**禁忌**：避免现代电子合成器质感、金属摇滚、恐怖阴森音效  
**整体感觉**：像在仙境花园里的小精灵在玩耍，清脆悦耳、轻盈灵动

---

## 技术要求

| 项目 | 要求 |
|------|------|
| 格式 | MP3（≤128kbps）或 WAV（16bit/44.1kHz） |
| 时长 | 短音效 0.1~0.5 秒，中等音效 0.5~1.5 秒，长音效 1~3 秒 |
| 体积 | 单个文件尽量 ≤50KB（短音效），BGM ≤2MB |
| 采样率 | 44100Hz |
| 声道 | 单声道（Mono），BGM 可立体声 |
| 响度 | 统一标准化到 -12dB LUFS 左右，避免音量差异过大 |

> **重要**：部分音效会被程序变速播放（0.5x~2.0x），生成时请确保原始音高在**中频区域（C4~C5）**，这样变速后高低音区都不会失真。

## 一、背景音乐（2个文件）

### 1. bgm.mp3 — 主背景音乐
- **文件名**：`bgm.mp3`
- **时长**：60~90 秒，**无缝循环**
- **风格**：轻快活泼的国风小调，可爱灵动
- **乐器**：古筝拨弦为主旋律 + 竹笛点缀 + 轻柔鼓点节拍 + 风铃装饰音
- **节奏**：中速偏快（BPM 110~130），适合消除游戏的轻松节奏
- **情绪**：愉悦、放松、不紧张，像在仙境草地上散步
- **注意**：循环接点要自然，头尾音量渐变衔接

### 2. boss_bgm.mp3 — Boss 战背景音乐
- **文件名**：`boss_bgm.mp3`
- **时长**：60~90 秒，**无缝循环**
- **风格**：紧张激昂的国风战斗曲，但保持可爱底色（不要太黑暗）
- **乐器**：大鼓/战鼓节拍 + 琵琶急速弹奏 + 古筝重音 + 笛子高音穿插
- **节奏**：快速（BPM 140~160），营造紧迫感
- **情绪**：紧张刺激但不恐怖，像萌系角色在认真对战

## 二、棋盘交互音效（5个文件）⭐ 核心体验

> 这组是玩家操作最频繁听到的音效，**直接决定游戏手感**，请重点打磨。

### 3. combo.mp3 — 连击音 ⭐最核心
- **文件名**：`combo.mp3`
- **时长**：0.15~0.25 秒
- **音色**：古筝/琵琶单弦拨弦音，清脆悦耳，像玉珠落盘
- **音高**：C4（中央C），因为程序会用 playbackRate 将其变速到 Do~Do' 八度音阶
- **衰减**：快速自然衰减，干净利落无拖尾
- **关键词**：清脆、干净、悦耳、有弹性
- **AI 提示词参考**：`古筝单弦拨弦，清脆如玉珠落盘，C4音高，短促0.2秒，快速衰减，无混响，干净明亮`

### 4. eliminate.mp3 — 珠子消除音 ⭐重要
- **文件名**：`eliminate.mp3`
- **时长**：0.2~0.35 秒
- **音色**：玉石/水晶碎裂的清脆声 + 一点点仙气回响，像泡泡破裂但更有质感
- **特点**：起音干脆（有明确的"碎"的瞬态），尾音带一丝空灵回响
- **音高**：中高频（E4~G4），明亮但不刺耳
- **关键词**：清脆碎裂、灵动、满足感
- **AI 提示词参考**：`玉石轻碎声混合水晶铃音，明亮清脆，0.3秒，起音尖锐尾音空灵，像修仙世界的灵珠消散`

### 5. rolling.wav — 珠子移动/交换音

- **音色**：两颗圆润玉珠轻轻碰撞滑动的声音，柔和短促

## 音效关系与层叠说明

> 以下信息帮助理解音效之间的配合关系，生成时请注意音色的**协调性**。

### 连击递进系统
`combo.mp3` 是核心，程序会将其变速播放构成 Do-Re-Mi-Fa-Sol-La-Si-Do 八个音阶。因此：
- `combo.mp3` 必须是**干净的单音**，不要有复杂和弦或噪声
- 以 C4 为基准音高，程序变速范围 1.0x~2.0x

### 消除层叠
- 3消：`eliminate.mp3` 单独播放
- 4消：`eliminate.mp3` + `combo.mp3` 叠加
- 5消：`eliminate.mp3` + `skill.wav` + `combo.mp3` 三层叠加

请确保这三个音效叠加时**不会打架**——频率分布要有差异：
- `eliminate.mp3`：中高频（碎裂感）
- `combo.mp3`：中频（拨弦感）
- `skill.wav`：中高频偏高（气流感）

### 里程碑和弦
5连/8连/12连时，`levelup.wav` + `combo.mp3` + `eliminate.mp3` 会同时播放构成和弦。请确保这三个音效的音色**风格统一**，能形成和谐的叠加效果。
