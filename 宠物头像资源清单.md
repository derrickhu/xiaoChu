# 宠物头像资源清单（100只）

## 统一画风说明

**画风参考**：**Q版/SD（Super Deformed）2D卡通风格** — 类似仗剑传说（Archero）、原神Q版立绘、崩坏星穹铁道Q版的风格。大头大眼小身体比例（头身比约2:1~3:1），粗黑色描边、高饱和度纯色块填充、**完全平涂无光影**、简洁可爱的卡通渲染。融入**中国仙侠神话**元素（道袍、法衣、祥云、仙灵兽、灵纹、玉饰等），**严禁**出现西幻风格（板甲、翅膀头盔、十字架、龙翼骑士等），**严禁写实风格**。

**构图规范**：1:1正方形头像，仅显示**大头 + 肩部**（Q版大头比例，头部占画面70%以上），纯色背景（便于抠图），不画全身。

**配色原则**：
- 角色整体配色**不强制使用属性对应颜色**，应以角色名称和性格特征为主导，使用**鲜艳、饱和、有区别度**的多样化色彩
- 属性元素仅通过**局部细节**体现，例如：手持物品、头饰、花纹纹理、眼睛颜色、饰品等某一个或几个细节点缀即可
- 同属性20个角色之间色彩应有**明显区分度**，避免全部用同一色系
- 参考仗剑传说（Archero）的鲜艳多彩画面风格

**抠图友好约束（重要）**：
- **必须有清晰的粗黑色描边轮廓**，所有颜色严格限制在描边线条内部，描边外侧不得有任何颜色溢出
- **禁止使用渐变特效、模糊特效、辉光/发光特效、光晕、柔光、阴影**等会导致边缘模糊的效果
- **禁止半透明、羽化边缘、光芒扩散、写实光影**等超出角色轮廓描边的视觉效果
- 背景必须为**纯色（solid color）或透明背景**，不使用渐变背景
- 角色边缘必须干净锐利，确保可以轻松抠图

**通用 Prompt 前缀**（所有头像共用）：
```
chibi 2D cartoon game character icon, super deformed cute style, big head small body ratio 2:1, thick clean black outlines, flat cel-shaded coloring with NO shading NO shadow NO gradient, vibrant saturated solid color blocks, colorful diverse palette, Chinese Xianxia mythology aesthetic, NOT Western fantasy, NOT realistic, NOT 3D, NOT semi-realistic, square 1:1 aspect ratio, close-up showing oversized head and tiny shoulders only, solid plain single-color background, big expressive cartoon eyes, game icon style, no text, no watermark, no glow effects, no gradient, no blur, no soft edges, no lighting effects, no color bleeding outside outlines, all colors strictly inside outlines, clean sharp edges for easy cutout, sticker-style clean border
```
