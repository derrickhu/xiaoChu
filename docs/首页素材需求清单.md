## 生成后处理流程

1. 使用 Midjourney v6 / DALL-E 3 / Stable Diffusion 生成
2. **场景背景**：裁剪到 750×1334，保存为 JPG（质量 90%），直接替换 `assets/backgrounds/` 下对应文件
3. **卡片背景**: 调整透明度，确认圆角与代码中 `12*S` 圆角一致，裁剪到 500×400
4. **导航图标**: 去除绿色背景（Photoshop / remove.bg），导出 96×96 透明底 PNG
5. **小号按钮**: 去除绿色背景，导出 200×60 透明底 PNG
6. 放置到对应目录后通知开发接入
