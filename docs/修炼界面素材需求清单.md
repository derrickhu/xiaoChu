## A. 修炼洞府背景图

- **文件名**: `cultivation_bg.png`
- **尺寸**: 750 × 1334 px（竖屏手机比例 9:16）
- **格式**: PNG 或 JPG
- **存放路径**: `assets/backgrounds/cultivation_bg.png`
- **用途**: 修炼界面全屏背景，UI 元素（头像、星盘节点、文字）会叠加在上面

### 设计要点

1. **洞府场景但保持简洁**：一个宽敞的山中洞府内部，岩壁圆润柔和（Q版风格），地面有一块浅色圆形石台（暗示打坐修炼之处），但不要太多杂物
2. **色调与首页一致**：暖米白为主色，搭配浅粉、淡紫、柔金色调。岩壁用淡灰褐色水墨笔触，不要深色暗调
3. **灵气元素**：几缕浅金色/淡青色灵气如丝带飘浮，2-3 个小光点散落空中，营造灵气充盈氛围
4. **小灵兽点缀**（重要）：画面边缘或角落安排 2-3 只小灵兽增加代入感：
   - 右下角蹲着一只打盹的小猫灵（类似金锋灵猫，Q版可爱）
   - 左侧岩壁上趴着一只小灵鸟
   - 或一只小狐狸从洞口探头
5. **构图留白**：画面中央和上半部分需大面积留白/浅色区域，因为 UI 内容（头像+星盘+文字）会覆盖在此处
6. **洞口透光**：洞府顶部或侧面有一处透光口，柔和的暖光洒入，增添温馨感

## B. 打坐角色形象（核心素材 ⭐）

修炼界面中央不再使用圆形头像裁剪，而是显示**完整的打坐全身小人**，更有代入感。
玩家可在"选择形象"面板中切换不同角色外观，不同境界还会叠加灵气特效。

### 统一规格

- **尺寸**: 512 × 512 px（正方形，角色居中）
- **格式**: PNG（**必须透明底**，代码直接叠加在洞府背景上）
- **存放路径**: `assets/hero/char_{id}.png`
- **用途**: 修炼界面中央打坐位，角色会有轻微上下悬浮动画

### 统一画风要求

- **整体水墨国风**，但各角色可在此基础上自由发挥风格差异（Q版/半写实/精致水墨均可）
- **盘腿打坐姿势**（双腿盘坐、双手放膝上或结印）
- 全身入画，**必须正面朝前**（不要侧身、不要 3/4 侧面）
- **表情和手势各角色自由差异化**（不要全部闭眼微笑），具体见各角色描述
- ⚠️ **头身比约束（最重要！）**：
  - 允许范围：**1:2 ~ 1:3**（头占整体约 1/3 ~ 1/4）
  - **1:2**（萌系 Q 版）：头大身小，适合可爱型角色（如修仙少年、天罡道童）
  - **1:3**（精致少年）：头部仍偏大但身体比例更修长，适合帅气/优雅角色（如剑灵少侠、星月仙子）
  - **绝对不能超过 1:3**，不要 1:5 或 1:6 那种写实小头长身，否则和其他角色放在一起大小不协调
  - 参考标准：打坐姿态下，头部宽度 ≈ 肩膀宽度的 70-100%（Q版接近100%，精致型约70-80%）
- **细线描边**（不要粗黑描边），线条精致清晰
- **人物周围不要白边**（不要 sticker 风格的外轮廓白描边）
- **人物脚下/座下不要任何光效、光圈、莲台、蒲团等物体**，只要人物本身悬空坐着
- **纯色背景**（推荐纯绿 `#00FF00` 方便抠图），或透明底。不要棋盘格、不要渐变
- 不要地面阴影，不要任何底座

CRITICAL PROPORTION RULE: head-to-body ratio MUST be between 1:2 and 1:3 (the head
including hair should be 1/3 to 1/4 of total character height), 1:2 for cute Q-style

NEVER use realistic slim small-head proportions like 1:5 or 1:6, head width should
be 70-100% of shoulder width, all characters must stay within this range for visual

## C. 头像图（用于选择面板预览和其他 UI 场景）

每个角色形象对应一个头像图，用于形象选择面板的缩略图预览以及其他需要头像的地方。
如果打坐图效果好，面板中也会直接显示打坐图缩略，头像图为备用。

- **尺寸**: 512 × 512 px
- **格式**: JPG（质量 85%）
- **存放路径**: `assets/hero/avatar_{id}.jpg` 或 `hero_cultivation.jpg`（boy1）
- **构图**: 大头 + 肩部（头部占 70%+），1:1 正方形
- **画风**: 与打坐图完全一致的角色面部特写

> 头像提示词 = 打坐图提示词中的角色描述部分 + 替换姿势为 "close-up head and tiny shoulders only"

## 注意事项

1. **打坐图抠图要点**：
   - 推荐生成时用纯绿色（#00FF00）背景，生成后用工具去除绿背转为透明底 PNG
   - **人物周围不要有白色描边/白边/sticker轮廓**，边缘要干净融入背景
   - **人物座下不要有任何光效、光圈、莲台、蒲团、底座**，必须是纯悬空坐姿
   - 最终导出为 PNG 透明底，代码直接叠加在洞府背景图上
2. **风格可以多变**：各角色可以有不同程度的风格差异（从 Q 版到精致水墨半写实），但整体保持水墨国风基调
3. **细线描边**：不要粗黑描边，用精细的墨线勾勒，线条清晰但不厚重
4. 背景图已转换为 JPG（85% 质量），当前尺寸合适，不需要暗化处理
5. 属性图标生成后建议统一裁剪为 128×128，确保圆形裁剪后居中美观
6. 打坐图是最高优先级素材，先生成 `char_boy1.png`（默认形象）即可调试
7. 锁定形象（boy3、girl3）当前标记为 `unlocked: false`，后续可通过成就/奖励解锁
8. 头像图优先级最低，因为选择面板中也会显示打坐图缩略。仅在其他 UI 场景需要时再生成
