## 统一风格说明

现有素材风格：**国风水墨/淡彩 + 金色祥云装饰边框**
- 主色调：暖金色、浅墨色、淡紫粉色云雾
- 边框/装饰：金色描边 + 祥云纹样（参考 `assets/ui/btn_start.png`）
- 背景融合：与 `assets/backgrounds/home_bg.jpg` 的水墨塔/五行阵/云山场景自然衔接

### 所有提示词共用风格锚定前缀

```
Chinese traditional ink wash watercolor game UI asset, light and warm color palette,
soft cream/warm beige tones like rice paper texture, pastel pink and lavender cloud
accents, ornamental gold filigree borders with auspicious cloud (祥云) motifs,
```

## 生成后处理流程

1. 使用 Midjourney v6 / DALL-E 3 / Stable Diffusion 生成
2. **卡片背景**: 调整透明度，确认圆角与代码中 `12*S` 圆角一致，裁剪到 500×400
3. **导航图标**: 去除绿色背景（Photoshop / remove.bg），导出 96×96 透明底 PNG
4. **小号按钮**: 去除绿色背景，导出 200×60 透明底 PNG
5. 放置到对应目录后通知开发接入
